Installer teensy loader pour pi.

copier 
00-teensy.rules et teensy_raspberrypi.tar.gz dans le dossier document du pi

puis utiliser ces commandes

cd Documents/
sudo cp 00-teensy.rules /etc/udev/rules.d/
tar -xvzf teensy_raspberrypi.tar.gz

Puis pour lancer le loader

./teensy &

Choisir le fichier hex puis telecharger et redemarrer
